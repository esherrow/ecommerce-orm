  # Installation
  * npm install,
  * npm install sequelize, 
  * npm install express, 
  * npm install mysql2, 
  * npm i dotenv.  
  * In MySQL: 
  * * source schema.  
  * On Command Line: 
  * * npm run seed.
  
  # Usage
  npm start
